Nesse teste vou utilizar todos os metodos aprendidos no HTML5

Negrito / Itálico

Para deixar o texto em Negrito usamos a tag < strong > entre o texto que queremos usar
E para deixar o texto em Itálico usamos a tag < em > entre o texto que queremos usar.

Quebra de linha

Para quebrar linha é necessario usar a tag < br >
Como nesse exemplo

Icone de Favoritos

Adicionando o " link:favicon " na tag < head > podemos escolher qual imagens vai ser usanda no icone de favoritos

Imagem

Usando a tag < img >, conseguimos acrescentar imagens no nosso site
E usando height e width podemos alterar o tamanho da imagem

Código-fonte / Pré-formatação

Usamos a tag < code > para citar um comando document.getElementById('teste') como nesse exemplo

Juntamente podemos usar a tag < pre > para o texto considerar a identação feita no codigo. Exemplo:

num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é PAR')
else:
    print(f'O número {num} é IMPAR')
print('Fim do programa')

Citação simples

Podemos usar a tag < q > para informar que o texto se trata de uma citação. Exemplo: o computador é uma burro muito rapido
